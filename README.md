# dcinsideCollector

### 사용법
------------

실행 시 아래와 같은 화면이 생성됩니다.

![main](https://user-images.githubusercontent.com/68284806/160861342-1a4591ff-33d2-4ac5-8be6-7eb8986f5403.png)

상단의 "갤러리 불러오기" 버튼을 클릭합니다.


그러면 아래와 같은 갤러리 찾기 창이 생성됩니다.

![galleryFind](https://user-images.githubusercontent.com/68284806/160861713-36c6621b-4340-4ee4-a4e4-a8c5b6f9dcf1.png)

이 창에서 갤러리 ID를 입력합니다.

갤러리 ID는 갤러리 주소에서 ?id= 부분에 있는 문자열입니다.

### 예시
![urlExample](https://user-images.githubusercontent.com/68284806/160862168-6fd8bf6c-1124-4af1-ab9b-a37ccc4bcc83.png)

싱글벙글 지구촌 갤러리으이 경우 ID는 `singlebungle1472`입니다.

---------

![gallSearchExample](https://user-images.githubusercontent.com/68284806/160862406-c7b8cec5-13c5-41e7-8425-62281ec6ff92.png)

갤러리 ID를 입력 후 `검색` 또는 `Enter`를 누르면 정보가 표시됩니다.

그 후 최대 조회 페이지 수와 탐색 범위를 지정합니다.

최대 페이지를 `0`으로 설정 시 무제한으로 설정됩니다.

정보를 입력한 후 `확인`을 누릅니다.

![saveHtmlButton](https://user-images.githubusercontent.com/68284806/160862876-e98076a1-b183-40e6-9e48-5e4082f4597f.png)

메인 화면으로 다시 돌아온 후 글 목록이 모두 불러와졌다면 우측 상단의 `HTML 출력`을 클릭합니다.

그 후 HTML로 저장하여 파일을 브라우저로 연 뒤 우측 클릭하여 `페이지 소스 보기`를 누릅니다.

페이지가 원본 소스로 열렸다면 `Ctrl+A` 버튼을 눌러 전체 선택 한 뒤 복사합니다.

![wrtieArticle](https://user-images.githubusercontent.com/68284806/160863348-2cc4d7ac-1f36-43d0-968e-f3cefbbe55a3.png)

디시인사이드에서 글 쓰기를 누른 뒤 우측 상단 표시된 HTML 체크 박스를 선택하면 HTML 모드로 전환됩니다.

이 곳에 복사한 소스를 붙여넣기 한 뒤 HTML 체크 박스를 해제하면 보기 쉽게 변환됩니다.
